# Changelog

HPB分析ツールの変更履歴

## [1.4.0] - 2024-12-17

### Added
- 予約比率データ（男女比・年齢層）の取得機能
  - HPBページの「予約比率」セクションからデータを自動取得
  - 性別比率を円グラフで可視化
  - 年代比率を横棒グラフで可視化
- PDF/テキストレポートに予約比率セクションを追加

### Changed
- 価格表示から「税込」表記を削除（HPBは全て税込価格のため）
- チェック項目3-4を「追加料金・オプション料金の明記（価格透明性）」に変更

## [1.3.0] - 2024-12-17

### Added
- PDFレポートにレーダーチャート・バーチャートを追加
- 採点詳細（チェック項目）をPDFに追加

### Changed
- PDFレポートのレイアウト改善

## [1.2.0] - 2024-12-17

### Fixed
- Streamlit Cloud環境でのPDFフォント問題を修正
- テキストダウンロードボタンのスタイル修正

## [1.1.0] - 2024-12-17

### Added
- PDFダウンロード機能
- テキストレポートダウンロード機能

### Changed
- 用語を「集客力」「予約力」に統一

## [1.0.0] - 2024-12-17

### Added
- 初回リリース
- Gemini API URL Contextによるサロンページ分析
- 4項目スコアリング（集客力、予約力、価格競争力、差別化）
- 競合比較機能（最大2店舗）
- レーダーチャート・バーチャートによる可視化
- AIチャット機能
- YouTube動画連携（美容サロン経営カレッジ）
- レスポンシブUI（スマホ対応）
